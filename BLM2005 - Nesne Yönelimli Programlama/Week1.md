# Nesne yönelimli Proogramlamaya Giriş

```java
class Circle{

    double radius = 1.0;

    Circle(){
        // Constructor (if no constructor created creates itself)
    }

    Circle(double newRadius){
        radius = newRadius;
    }

    double getArea(){
        return radius * radius * 3.14159;
    }
}
```
